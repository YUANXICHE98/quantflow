# 因子库

## 1. MACD 指标 (`factor-macd`)

### 基本信息
- **名称**: MACD 指标
- **类型**: 趋势因子
- **描述**: MACD 指数平滑异同移动平均线，用于判断趋势方向和动量变化
- **Chat 引用**: `@factor:macd`

### 参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| fastPeriod | number | 12 | 5-20 | 快线 EMA 周期 |
| slowPeriod | number | 26 | 15-50 | 慢线 EMA 周期 |
| signalPeriod | number | 9 | 5-20 | 信号线周期 |

### 输入
```json
{
  "prices": [100, 102, 101, ...],
  "fastPeriod": 12,
  "slowPeriod": 26,
  "signalPeriod": 9
}
```

### 输出
```json
{
  "macd": 2.5,
  "signal": 1.8,
  "histogram": 0.7,
  "crossType": "golden",
  "trend": "bullish"
}
```

### 信号规则

| 条件 | 动作 | 说明 |
|------|------|------|
| MACD 上穿信号线 | buy | 金叉买入信号 |
| MACD 下穿信号线 | sell | 死叉卖出信号 |
| 柱状图 > 0 | bullish | 多头动能增强 |
| 柱状图 < 0 | bearish | 空头动能增强 |

## 2. RSI 极端值因子 (`factor-rsi-extreme`)

### 基本信息
- **名称**: RSI 极端值因子
- **类型**: 动量因子
- **描述**: 检测 RSI 超买(>70)和超卖(<30)信号
- **Chat 引用**: `@factor:rsi-extreme`

### 参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| period | number | 14 | 5-50 | RSI 计算周期 |
| overbought | number | 70 | 60-90 | 超买阈值 |
| oversold | number | 30 | 10-40 | 超卖阈值 |

### 输出
```json
{
  "value": 78.5,
  "signal": "sell",
  "strength": 60,
  "description": "RSI 超买区域 (78.5 > 70)"
}
```

### 信号规则

| 条件 | 信号 | 说明 |
|------|------|------|
| RSI < 30 | buy | 超卖，关注做多机会 |
| RSI > 70 | sell | 超买，关注做空机会 |
