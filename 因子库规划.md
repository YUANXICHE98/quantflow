# QuantFlow 因子库/算法库/模型库规划

## 一、因子库规划

### 1. 技术因子（基于量价数据）

| 因子名称 | 类型 | 说明 | 实现难度 |
|---------|------|------|---------|
| RSI | 动量 | 相对强弱指数，超买超卖判断 | ⭐ |
| MACD | 趋势 | 异同移动平均线，金叉死叉 | ⭐ |
| 布林带 | 波动率 | 价格通道突破 | ⭐ |
| KDJ | 动量 | 随机指标 | ⭐ |
| MA均线 | 趋势 | 多周期均线交叉 | ⭐ |
| ATR | 波动率 | 真实波动幅度 | ⭐⭐ |
| OBV | 成交量 | 能量潮指标 | ⭐⭐ |
| VWAP | 成交量 | 成交量加权均价 | ⭐⭐ |
| 威廉指标 | 动量 | WR超买超卖 | ⭐ |
| CCI | 趋势 | 顺势指标 | ⭐ |

| 因子名称 | 数据源 | 说明 |

| 资金流向 | 交易所数据 | 主力资金进出 |
| 合约持仓 | 交易所数据 | 多空比、持仓量变化 |

### 3. 链上数据因子（加密货币专用）

| 因子名称 | 说明 |
|---------|------|
| 链上转账量 | 大额转账监控 |
| 交易所流入流出 | 鲸鱼动向 |
| 活跃地址数 | 网络活跃度 |
| 挖矿难度 | 算力变化 |

## 二、算法库规划

### 1. 趋势跟踪策略

```
策略名称: 双均线交叉
入场: 短期均线上穿长期均线
出场: 短期均线下穿长期均线
参数: short_window=5, long_window=20
```

```
策略名称: 海龟交易
入场: 价格突破20日最高价
出场: 价格跌破10日最低价
参数: entry_window=20, exit_window=10
```

### 2. 均值回归策略

```
策略名称: 布林带均值回归
入场: 价格触及下轨
出场: 价格回归中轨
参数: window=20, num_std=2
```

```
策略名称: RSI超卖反弹
入场: RSI < 30
出场: RSI > 70
参数: period=14
```

### 3. 统计套利策略

```
策略名称: 配对交易
原理: 两只相关性高的股票价差回归
入场: 价差超过2倍标准差
出场: 价差回归均值
```

```
策略名称: 协整套利
原理: 利用协整关系的长期均衡
实现: 需要协整检验 (ADF test)
```

### 4. 网格交易策略

```
策略名称: 等差网格
原理: 在价格区间内等间距挂单
参数: grid_num=10, price_range=[low, high]
适用: 震荡行情
```

```
策略名称: 等比网格
原理: 按比例设置网格间距
参数: grid_ratio=1.02
适用: 单边上涨/下跌行情
```

### 5. 动量策略

```
策略名称: 价格动量
入场: 过去N日收益率 > 阈值
参数: lookback=20, threshold=0.1
```

```
策略名称: 成交量动量
入场: 成交量放大 + 价格上涨
参数: volume_ratio=2.0
```
