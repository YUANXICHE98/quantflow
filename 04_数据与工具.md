# 数据与工具

## MCP 数据源

### OKX 行情数据 (`okx-data`)

| 属性 | 值 |
|------|-----|
| 类型 | MCP |
| 名称 | OKX 行情数据 |
| 描述 | 实时行情、历史 K 线数据 |
| 状态 | 已接入 |

**支持数据**:
- 实时行情
- 历史 K 线
- 合约数据

## Skill 技能

### 飞书通知 (`feishu-notify`)

| 属性 | 值 |
|------|-----|
| 类型 | Skill |
| 名称 | 飞书通知 |
| 描述 | 推送信号到飞书群 |
| 状态 | 已接入 |

**使用场景**:
- 交易信号推送
- 策略执行通知
- 风险预警
