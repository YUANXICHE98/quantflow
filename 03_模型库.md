# 模型库

## LSTM 价格预测模型 (`model-lstm-predictor`)

### 基本信息
- **名称**: LSTM 价格预测模型
- **类型**: 时序预测
- **描述**: 基于长短期记忆网络的价格走势预测模型
- **Chat 引用**: `@model:lstm-predictor`

### 架构配置

| 参数 | 值 |
|------|-----|
| 类型 | LSTM |
| 输入特征 | close, volume, high, low |
| 序列长度 | 60 |
| 隐藏层 | [128, 64] |
| 输出维度 | 1 |
| Dropout | 0.2 |

### 训练配置

| 参数 | 值 |
|------|-----|
| 学习率 | 0.001 |
| 训练轮数 | 100 |
| 批大小 | 32 |
| 优化器 | Adam |
| 损失函数 | MSE |

### 输入
```json
{
  "symbol": "BTC",
  "timeframe": "1h",
  "sequenceLength": 60
}
```

### 输出
```json
{
  "prediction": 45250.5,
  "confidence": 85,
  "direction": "up",
  "timestamp": "2026-04-09T10:00:00Z"
}
```

### 性能预期

| 指标 | 值 |
|------|-----|
| 预期准确率 | 88-95% |

### 后端要求

- **类型**: Python 后端
- **端点**: `/api/models/lstm/predict`

### 降级方案

当 LSTM 后端不可用时，使用简单移动平均预测。
